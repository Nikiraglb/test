1. Тест
Цель тестирования: Проверить, что метод markCompleted корректно помечает задачу как выполненную.

Условия и предпосылки:

Создан экземпляр класса Task.

Шаги тестирования:

Создать экземпляр класса Task с названием "Test Task".

Вызвать метод markCompleted у созданного экземпляра.

Ожидаемый результат: Свойство completed задачи должно быть true.

2. Тест
Цель тестирования: Проверить, что метод addTask добавляет задачу в список задач.

Условия и предпосылки:

Создан экземпляр класса TodoList.

Создана задача класса Task.

Шаги тестирования:

Создать экземпляр класса TodoList.

Создать задачу Task с названием "Test Task".

Добавить задачу в список с помощью метода addTask.

Ожидаемый результат: Задача должна быть в списке задач.

3. Тест

Условия и предпосылки:

Создан экземпляр класса TodoList.

Создана и добавлена задача в список.

Шаги тестирования:

Создать экземпляр класса TodoList.

Создать задачу Task с названием "Test Task".

Добавить задачу в список с помощью метода addTask.

Удалить задачу из списка с помощью метода removeTask.

Ожидаемый результат: Задача не должна быть в списке задач.

4. Тест
Цель тестирования: Проверить, что метод editTask заменяет старую задачу новой задачей.

Условия и предпосылки:

Создан экземпляр класса TodoList.

Создана и добавлена задача в список.

Шаги тестирования:

Создать экземпляр класса TodoList.

Создать задачу Task с названием "Test Task".

Добавить задачу в список с помощью метода addTask.

Создать новую задачу Task с названием "New Task".

Заменить старую задачу новой с помощью метода editTask.

Ожидаемый результат: Список задач должен содержать новую задачу, но не должен содержать старую.

5. Тест
Цель тестирования: Проверить, что метод getAllTasks возвращает все задачи из списка.

Условия и предпосылки:

Создан экземпляр класса TodoList.

Создана и добавлена задача в список.

Шаги тестирования:

Создать экземпляр класса TodoList.

Создать задачу Task с названием "Test Task".
